# Zomato Analysis Dashboard using Power BI

This project presents an interactive Power BI dashboard developed to analyze Zomato restaurant data and uncover valuable business insights related to customer behavior, restaurant performance, revenue generation, delivery efficiency, and order cancellations.

The dashboard includes multiple analytical views such as Overall Analysis, Restaurant Analysis, Revenue Analysis, Delivery Analysis, Customer Ratings Analysis, and Cancellation Analysis. Through the use of KPIs, charts, filters, and drill-down features, the project enables users to monitor key metrics and make data-driven decisions.

Key insights include identifying top-performing restaurants, analyzing customer satisfaction trends, tracking revenue patterns, evaluating delivery performance, and understanding cancellation reasons and their impact on business operations.

This project demonstrates proficiency in Power BI concepts including data transformation using Power Query, data modeling, DAX calculations, interactive visualizations, and dashboard design for business intelligence and reporting.
